About This Project

This project demonstrates advanced CSS styling based on a Figma design file. It uses custom fonts, colors, and layout techniques.

Features

  • Custom fonts (Source Sans Pro, Spin Cycle OT)
  • Responsive design with Flexbox and Grid
  • Styled header and footer
  • Use of images and icons

How to Use

Open the index.html file in your browser to see the styled page. Resize the window to see responsive behavior.
